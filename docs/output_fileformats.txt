Formats of output files
=======================


Path: ~/ibm_vessels/DISPLACE_outputs/(basepath)/(basename)/

* vmslike_simu2.dat
-------------------
Written by: main()
See: simulator/main.cpp
Content: Position of each vessel, if state != 3
Format:
tstep name tstep_dep x y course cum_fuel state

* popstats_*.dat
----------------------
Written by: biomodule.cpp
See:
Content:
Format:
tstep / pop / N sz0 / N sz1 /.../ F age0/ F age1 /.../ SSB sz0/ SSB sz1/...
Collected Fields:




* popnodes_start_*.dat
----------------------
Written by: Node::export_popnodes()
See: commons/Node.cpp
Content:
Format:
tstep / node / long / lat / tot N sp0 / tot W sp0/ tot N sp1 / tot W sp1...
Collected Fields:

* popnodes_inc_*.dat
--------------------
See popnodes_start_*.dat

* popnodes_impact_*.dat
-----------------------
Written by: Node::export_popnodes_impact()
See: commons/Node.cpp
Content:
Format:
pop / tstep / node_idx / long / lat / impact_on_pop
Collected Fields: (0,1,2,)5


* popnodes_cumulcatches_per_pop_*.dat
-----------------------
Written by: Node::export_popnodes_cumulcatches_per_pop()
See: commons/Node.cpp
Content:
Format:
pop / tstep / node_idx / long / lat / cumcatches (actually landings)
Collected Fields: (0,1,2,)5


* popdyn_*.dat
--------------
Written by: Population::export_popdyn_N
See: commons/Population.cpp
Content: Aggregate number of populations. all for tstep=0, some selected for others. Number in thousands.
Format:
tstep name total_for_groups[0..n]

* popdyn_F_*.dat
----------------
See popdyn_*.dat

* popnodes_cumftime
--------------------
Written by: Node::export_popnodes()
See: commons/Node.cpp
Content:
Format:
tstep   node   long    lat    cumftime

* popnodes_cumsweptarea
--------------------
Written by: Node::export_popnodes()
See: commons/Node.cpp
Content:
Format:
tstep   node   long    lat    cumsweptarea subsurfacecumsweptarea

* loglike_*.dat
---------------
Written by: Vessel::export_loglike()
See commons/Vessel.cpp
Content: Vessel status as a logbook
Format:
0 tstep_dep
1 tstep         Current tstep
2 reason_to_go_back
3 cum_steam
4 loc->nodeidx (Harbour?)
5 idx
6 name
7 timeatsea
8 cumfulecons
9 travel_dist_this_trip
10xN    cumul sz per pop(1..N)
freq_metiers
revenue
revenue_from_av_prices
revenue_explicit_from_av_prices
fuelcost
gav
gav2 // gav handling energy cost only
sweptarea // from BENTHIS parameterisation
revenuepersweptarea
GVA // gva handling all costs (but need a deeper parameterisation)
GVAPerRevenue
LabourSurplus
GrossProfit
NetProfit
NetProfitMargin
GVAPerFTE
RoFTA
BER
CRBER
NetPresentValue
numTrips


N: can be calculated with NUMFIELDS - 16.
