Detailed Description

template<typename Operator>
class dtree::TwoArgumentsComparatorStateEvaluator< Operator >

A decorator class that evaluates the relation between the values of two other evaluators.

This class takes an operator as argument in the constructor, and uses it to operate on the two evaluators. A useful application is comparing two values. The following code snippet illustrate how to compare a double variable with a constant using the std::less functional class

#include <functional>
StateEvaluator *myEvaluator = new dtree::TwoArgumentsOperatorStateEvaluator<std::less<double> >(
new dtree::VariableReferenceStateEvaluator<double>(myvalue),
new dtree::ConstStateEvaluator<double>(1000.0),
std::less<double>());
